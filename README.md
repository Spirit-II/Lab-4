## Задание 1. Решить задачу на построение нового дерева по заданному (map)

### Текст задачи 

Дерево содержит строки. В конец каждой строки приписать ее же.

### Алгоритм решения

После создания дерева и проверки что оно не пустое, дублируем строку (предварительно изменив её тип данных) и рекурсивно запускаем этот же алгоритм ко всем потомкам узла.

## Задание 2. Решить задачу на получение заданного результата для указанного дерева (fold)

### Текст задачи 

Выяснить, содержится ли в дереве элемент с заданным значением.

### Алгоритм решения

Проверяем что дерево не пустое, после этого сравниваем значение узла со значением для проверки и в случае неудачи проверяем потомков узла по тому же алгоритму.
